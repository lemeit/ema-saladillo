EMA-EET · EEST N°1 "Gral. Savio" — Saladillo

Estación meteorológica automática marca Tecmes perteneciente a la red RMET del INA (Instituto Nacional del Agua). Transmite datos por telemetría al Sistema Nacional de Información Hídrica (SNIH) cada 1 hora. La torre de transmisión mide ~5-6 m con anemómetro y veleta en la cima. Los sensores de temperatura y humedad se ubican a ~1.2 m del suelo en escudo solar multipared. La estación se encuentra en el predio de educación física de la escuela, próxima a los paredones perimetrales.

Estación Davis Instruments administrada por el Centro de Formación Rural de Saladillo. Ubicada en zona periurbana, en la transición entre el tejido urbano y el área rural. EMA-DC · Defensa Civil — Aeródromo Provincial Saladillo

Estación Davis Instruments con transmisión vía Meteobridge, ubicada en el aeródromo provincial de Saladillo — zona rural abierta, sin obstrucciones, con buena exposición al viento. Es la estación más representativa de las condiciones rurales del partido. Los datos son transmitidos como overlay sobre imagen de cámara IP; se extraen mediante OCR. Actualiza cada ~5 minutos.

EMA-CS · Clima Saladillo — Barrio Falucho

Estación particular Davis Instruments administrada por un vecino del barrio Falucho. Publica datos en climasaladillo.com usando la plataforma Meteotemplate. Es la estación con mayor cobertura de parámetros: incluye temperatura interior, sensación térmica, evapotranspiración y punto de rocío. Los datos se obtienen a través del endpoint JSON nativo de Meteotemplate. El análisis comparativo muestra un efecto de isla de calor urbano nocturno de +2 a +4°C respecto a las estaciones periurbanas/rurales.

Nota metodológica
Todos los datos son operativos en tiempo útil, sin validación formal del organismo responsable. Las diferencias de temperatura entre estaciones reflejan tanto variabilidad microclimática real (exposición, cobertura del suelo, urbanización) como posibles offsets de calibración de los sensores. El análisis comparativo requiere series temporales prolongadas para distinguir ambos efectos.
Proyecto: Laboratorio de Industrias · 7° Año Técnico Químico · EEST N°1 "Gral. Savio" · Saladillo · 2026

📡 El proyecto

Red EMA Saladillo integra datos meteorológicos de 4 estaciones automáticas distribuidas en el partido de Saladillo, Buenos Aires. Los datos se recopilan cada hora, se almacenan en la nube y se visualizan en este portal en tiempo real.

El proyecto surge en el marco del espacio curricular Laboratorio de Industrias de 7° año del Técnico Químico, con el objetivo de establecer la infraestructura base para correlacionar variables meteorológicas con calidad del aire.

🎯 Objetivos

  • Acceder a datos meteorológicos en tiempo real de múltiples fuentes
  • Almacenar y visualizar datos desde cualquier dispositivo
  • Analizar gradientes espaciales y microclimáticos entre estaciones
  • Integrar sensores propios de calidad del aire
  • Generar materiales didácticos para el aula

🏗 Arquitectura técnica

  • Scheduler: GitHub Actions — cron horario

  • Dashboard: HTML/CSS/JS vanilla — single file
  • Hosting: GitHub Pages (gratuito)
